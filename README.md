# Project-Sanjaya
In the Mahabharata, Sanjaya was gifted Divya Drishti (divine vision). He could "retrieve" information from the distant battlefield and "generate" a live narrative for the blind King Dhritarashtra. The Engineering Metaphor: This is exactly what your application does. It uses a Vector DB (Divine Vision) to see data the LLM (the King) cannot see
